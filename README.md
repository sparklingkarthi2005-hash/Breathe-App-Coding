# BreatheIn Pro 🌿 — Ultimate Mental Wellness Micro-App :-

A lightweight , completely serverless mental wellness application engineered to combat digital fatigue , burnout , and *Screen Apnea* (unknowingly holding your breath or breathing shallowly during long working/coding sessions). 

By pairing advanced browser APIs with core neurological pacing , this zero-friction application helps users lower stress and restore mental clarity in under a minute .

## 🧠 Scientific & Neurological Foundation :-

Unlike standard breathing apps , **BreatheIn Pro** is structurally tailored around proven respiratory mechanics :-

* **Vagus Nerve Stimulation (VNS) :-** Implements a strict 4-4 breathing cadence (Box Breathing) that triggers the Parasympathetic Nervous System (PNS) to naturally regulate heart rate and lower acute anxiety .
* **Cortisol Management :-** Rhythmic breathing for just 60 seconds works as a biological circuit-breaker , reducing the production of cortisol (the primary stress hormone) .
* **Alpha Brainwave Shift :-** Smooth transition pacing encourages the brain to step down from hyper-focused/anxious Beta waves into high-retention, relaxed Alpha wave states .

## 🚀 Key Features :-

* **Procedural Sound Therapy 🎵 :-** Built using the **HTML5 Web Audio API** to synthesize calming low-frequency sine waves mathematically on the fly . No external audio files or streaming assets are required , ensuring zero media latency .
* **Customizable Pacing Modes ⏱️ :-** Features dynamic timer setups depending on current cognitive loads:
  * `1 Min` — Micro Reset (Quick break during debugging)
  * `3 Min` — Deep Calm (Mid-day realignment)
  * `5 Min` — Power Meditation (Deep relaxation)
* **Gamified Habit Tracker 🔥 :-** Implemented a client-side **Streak Tracker** via **LocalStorage** to track consecutive daily practice intervals seamlessly without remote tracking walls .
* **Adaptive Eye-Care Interface 🌗 :-** Includes a fully responsive Dark/Light mode toggle powered entirely by Tailwind utility layers to minimize blue-light exposure during late-night shifts .

## 🛠️ Built With (Tech Stack) :-

* **Frontend :-** Semantic HTML5 & Modern Utility-First Tailwind CSS (driven by dynamic transitions) .
* **Logic Framework :-** Vanilla JavaScript managing asynchronous `setInterval` execution loops , active DOM rendering states , and state-driven web audio processing .
* **Hosting & CI/CD :-** Version-controlled via Git/GitHub with immediate continuous integration pipelines hosted on **Render**.
